## 🎯 Обзор проекта

**Smart Curtain Control** — современное решение для автоматизации штор на базе ESP32 и Android. Проект представляет собой полноценную систему "умного дома" с интуитивно понятным интерфейсом и прямым управлением устройствами.

### 🏆 Ключевые особенности

- ⚡ **Прямое управление** — приложение общается с ESP32 напрямую через HTTP API
- 🔍 **Автоматическое обнаружение** — поиск устройств через mDNS (Bonjour)
- 📱 **BLE настройка Wi-Fi** — удобная первоначальная конфигурация через Bluetooth
- 💾 **Сохранение устройств** — список устройств сохраняется и не теряется после перезапуска
- 🎨 **Современный дизайн** — Material Design с анимациями и плавными переходами

## 🚀 Установка и настройка

### 📱 Android приложение

#### 1. Клонирование репозитория

```bash
git clone https://github.com/MallaboDeCherry/stori0.7.1.git
cd stori0.7.1
2. Открытие в Android Studio
Откройте Android Studio

Выберите Open an Existing Project

Укажите путь к скачанному проекту

3. Сборка и установка
bash
./gradlew assembleDebug
./gradlew installDebug
или через Android Studio: Run → Run 'app'

🔧 ESP32 прошивка
1. Клонирование репозитория
bash
git clone https://github.com/MallaboDeCherry/SmartCurtain_ESP32.git
cd SmartCurtain_ESP32
2. Настройка PlatformIO
Убедитесь, что установлен PlatformIO (расширение для VS Code или отдельная IDE)

3. Настройка пинов в Config.h
cpp
// Пины для шагового двигателя
#define DIR_PIN     18    // Направление (TB6600 DIR-)
#define STEP_PIN    17    // Шаговый импульс (TB6600 PUL-)
#define ENABLE_PIN  19    // Включение драйвера (TB6600 ENA-)

// Светодиод индикации
#define LED_PIN     2     // Встроенный LED
4. Сборка и прошивка
bash
pio run --target upload --target monitor
🌐 Первоначальная настройка ESP32 (BLE)
Включите ESP32 — он запустит BLE режим (светодиод быстро мигает)

Откройте приложение → нажмите + → Настройка нового ESP32 через BLE

Выберите ESP32 из списка найденных устройств

Введите данные вашей Wi-Fi сети (SSID и пароль)

ESP32 перезагрузится и подключится к вашей сети (светодиод горит постоянно при активном приложении)

🔍 Поиск ESP32 в приложении
В главном меню нажмите + → Поиск устройств в сети

Нажмите "Найти устройства" — приложение найдет ESP32 по mDNS

Выберите устройство — оно добавится в хаб

Нажмите на устройство для управления

🐛 Устранение неполадок
Проблема: ESP32 не находится в сети
Решение:

Убедитесь, что ESP32 включен и светодиод мигает/горит

Проверьте, что телефон и ESP32 в одной Wi-Fi сети

Откройте браузер и перейдите по адресу http://smart-curtain.local/ping

Если не работает, используйте прямой IP адрес ESP32

Проблема: Двигатель не вращается
Решение:

Проверьте питание TB6600 (12-24V)

Проверьте соединение катушек двигателя

Увеличьте STEP_DELAY в CurtainControl.cpp до 1000-1500

Проверьте настройки микрошага на TB6600

Убедитесь в общем GND между ESP32 и TB6600

Проблема: Приложение вылетает
Решение:

Очистите данные приложения в настройках телефона

Переустановите приложение

Проверьте разрешения (Интернет, Bluetooth, Местоположение)
