# 🚀 Быстрый старт - Настройка за 5 минут

## ✅ Что нужно сделать

### 1️⃣ Запустите Python сервер (30 секунд)

```bash
cd C:\Users\Hamehut\AndroidStudioProjects\sshtori\server
python curtain_server.py
```

**Результат:** Вы увидите сообщение "Запуск сервера на http://0.0.0.0:5000"

---

### 2️⃣ Узнайте IP адрес компьютера (30 секунд)

```bash
ipconfig
```

**Найдите строку:** `IPv4 Address. . . . . . . . . . . : 192.168.1.XXX`

**Запомните этот адрес!** Например: `192.168.1.100`

---

### 3️⃣ Запустите приложение на телефоне (10 секунд)

1. Откройте Android Studio
2. Нажмите ▶️ Run
3. Дождитесь запуска приложения

---

### 4️⃣ Настройте приложение (2 минуты)

1. **Нажмите кнопку ⚙️** (настройки) в правом нижнем углу

2. **Введите данные:**
   ```
   URL сервера:     http://192.168.1.100:5000
                    ↑ ваш IP адрес из шага 2

   Название штор:   Моя штора
   ID штор:         curtain_1
   IP NodeMCU:      192.168.1.101
   Порт NodeMCU:    80
   ```

3. **Нажмите:** "🔌 Проверить соединение"

4. **Если всё ОК:** Нажмите "💾 Сохранить настройки"

---

### 5️⃣ Готово! Пользуйтесь приложением 🎉

**Доступные действия:**
- 📂 **Открыть** - открыть шторы
- 📁 **Закрыть** - закрыть шторы
- ⏸️ **Стоп** - остановить движение
- 🎚️ **Слайдер** - установить точную позицию
- **25% / 50% / 75%** - быстрые позиции

---

## ⚠️ Если что-то не работает

### Проблема: "Сервер не отвечает"

**Решение:**
1. Проверьте, что Python сервер запущен
2. Проверьте IP адрес (должен совпадать с `ipconfig`)
3. Убедитесь, что телефон и компьютер в одной WiFi сети

### Проблема: "Приложение вылетает"

**Решение:**
1. Перезапустите приложение
2. Нажмите "🔄 Сбросить к значениям по умолчанию"
3. Введите настройки заново

## 🎯 Минимальная конфигурация для теста

Если у вас **НЕТ ESP32**, можно протестировать только с Python сервером:

1. Запустите Python сервер
2. Настройте только URL сервера в приложении
3. Все команды будут работать в режиме симуляции

**Сервер будет симулировать работу штор без реального оборудования!**
