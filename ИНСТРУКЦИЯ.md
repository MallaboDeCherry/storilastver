## 🚀 Пошаговая инструкция запуска

### Шаг 1: Запуск Python сервера

1. Откройте терминал/командную строку
2. Перейдите в папку сервера:
   ```bash
   cd C:\Users\Hamehut\AndroidStudioProjects\sshtori\server
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Запустите сервер:
   ```bash
   python curtain_server.py
   ```

5. Вы увидите:
   ```
   ============================================================
   Сервер управления умными шторами
   ============================================================
   Запуск сервера на http://0.0.0.0:5000
   NodeMCU32 адрес: http://192.168.1.101:80
   ============================================================
   ```

6. **Важно!** Запомните IP адрес вашего компьютера в локальной сети:
   - Windows: `ipconfig` (найдите IPv4 адрес)
   - Например: `192.168.1.100`

### Шаг 2: Настройка NodeMCU32 (ESP32)

1. Откройте Arduino IDE
2. Откройте файл `server/nodemcu32_example.ino`
3. Установите библиотеку ArduinoJson:
   - Sketch → Include Library → Manage Libraries
   - Найдите "ArduinoJson" и установите

4. Измените настройки WiFi:
   ```cpp
   const char* ssid = "ВАШ_WIFI";
   const char* password = "ВАШ_ПАРОЛЬ";
   ```

5. Укажите IP адрес компьютера с Python сервером:
   ```cpp
   const char* serverIP = "192.168.1.100";  // IP вашего компьютера
   ```

6. Подключите ESP32 к компьютеру
7. Выберите плату: Tools → Board → ESP32 Dev Module
8. Выберите порт: Tools → Port → COM...
9. Загрузите код: Sketch → Upload

10. Откройте Serial Monitor (115200 baud) и проверьте подключение

### Шаг 3: Запуск Android приложения

1. Откройте проект в Android Studio
2. Дождитесь синхронизации Gradle
3. Подключите Android устройство или запустите эмулятор
4. Нажмите Run (зеленая кнопка)

### Шаг 4: Настройка приложения

1. После запуска приложения нажмите на кнопку настроек (⚙️ внизу справа)
2. Введите данные:
   - **URL сервера**: `http://192.168.1.100:5000` (IP вашего компьютера)
   - **ID штор**: `curtain_1`
3. Нажмите "Сохранить"
4. Приложение автоматически проверит соединение

## 🔌 Подключение мотора к ESP32

```
ESP32 (NodeMCU32)    L298N Motor Driver
─────────────────    ──────────────────
GPIO 12         →    IN1
GPIO 14         →    IN2
GPIO 27 (PWM)   →    ENA
GND             →    GND
5V              →    +5V

L298N Motor Driver   DC Motor
──────────────────   ────────
OUT1            →    Motor +
OUT2            →    Motor -
```

**Схема подключения:**
1. Подключите ESP32 к драйверу мотора L298N
2. Подключите DC мотор к выходам драйвера
3. Подайте питание на драйвер (12V для мотора)
4. ESP32 питается от USB или отдельного 5V источника

## 🧪 Тестирование без NodeMCU

Можно протестировать приложение без реального ESP32:

1. Запустите только Python сервер
2. Сервер будет симулировать работу штор
3. Все команды будут обрабатываться, но не отправляться на ESP32
4. Состояние будет обновляться автоматически

## ❗ Решение проблем

### Приложение не подключается к серверу:

1. **Проверьте, что сервер запущен:**
   ```bash
   curl http://localhost:5000/api/ping
   ```
   Должен вернуть: `{"success":true,...}`

2. **Проверьте firewall:**
   - Windows: Разрешите Python в брандмауэре
   - Или временно отключите firewall для теста

3. **Проверьте IP адрес:**
   - Android устройство и компьютер должны быть в одной WiFi сети
   - Используйте правильный IP адрес компьютера

4. **Проверьте разрешения Android:**
   - Settings → Apps → Умные Шторы → Permissions
   - Должно быть разрешение на интернет

### ESP32 не подключается к WiFi:

1. Проверьте правильность SSID и пароля
2. Убедитесь, что WiFi 2.4GHz (ESP32 не поддерживает 5GHz)
3. Проверьте Serial Monitor для диагностики

### Мотор не работает:

1. Проверьте подключение проводов
2. Убедитесь, что драйвер L298N получает питание
3. Проверьте правильность пинов в коде
4. Используйте мультиметр для проверки напряжения
