## 🎮 Как использовать

### Шаг 1: Открыть настройки
```
Главный экран → Кнопка ⚙️ (правый нижний угол)
```

### Шаг 2: Изменить параметры
```
Введите свои значения в соответствующие поля
```

### Шаг 3: Проверить соединение
```
Нажмите "🔌 Проверить соединение"
```

### Шаг 4: Сохранить
```
Нажмите "💾 Сохранить настройки"
```

---

## 🔌 Кнопки в настройках

### 💾 Сохранить настройки
- Проверяет корректность данных
- Сохраняет все изменения
- Возвращает на главный экран

### 🔌 Проверить соединение
- Тестирует связь с сервером
- Показывает результат
- Не сохраняет настройки

### 🔄 Сбросить к значениям по умолчанию
- Восстанавливает все настройки
- Требует подтверждения
- Полезно при проблемах

### 📤 Экспорт настроек
- Показывает все настройки
- Копирует в буфер обмена
- Полезно для отладки
